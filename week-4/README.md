# Structred Outputs

## Requirements

- uv
- Python 3.12

## Setup

- Create a new directory: `week-4`
- Run `uv init -p 3.12 .`
    - `-p` defines the Python
- Run `uv venv -p 3.12`
- Enable the virtual environment: `source .venv/bin/activate`
- Install dependencies: `uv pip install openai pydantic`
